# 打小人 🥊

情绪释放小游戏 —— 点屏幕暴打小人，解压神器！

## 功能

- 👆 **点击即打** — 点击屏幕任意位置打一下，计数器 +1
- 🔁 **自动暴打** — 开启后 120ms/次 高速连打
- 💥 **打击特效** — 抖动动画 + 浮动文字 + 暴击粒子爆炸
- 🔊 **音效** — Web Audio API 合成打击音，无需外部文件
- 📸 **自定义脸图** — 上传图片替换小人脸部，仅存本地，不上传服务器
- 🔢 **连击检测** — 快速连打 3 下以上触发连击提示
- 💾 **数据持久化** — 打击次数和自定义图片均保存在 localStorage

## 技术栈

纯前端，单文件，无需构建：

- HTML + CSS + JavaScript
- Web Audio API（音效合成）
- localStorage（数据持久化）
- 零依赖，直接打开 `index.html` 即可运行

## 运行

直接用浏览器打开 `index.html`，或本地起一个静态服务器：

```bash
npx serve .
```
